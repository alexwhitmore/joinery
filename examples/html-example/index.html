Frequently asked questions

How is this different from Tailwind CSS?

Tailwind gives you utility classes for any CSS property. Joinery gives you semantic layout primitives. They work great together — use Joinery for structure, Tailwind for styling.

Do I need to use React?

No! Joinery is CSS-first. The React components are thin wrappers. You can use plain HTML with CSS classes, or any framework.

What browsers are supported?

All modern browsers. The CSS uses gap, clamp(), and CSS custom properties.

Can I customize the spacing tokens?

Absolutely. Override the CSS custom properties. The files live in your project.
